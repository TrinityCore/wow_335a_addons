Manual:

Copy the addon in your "yourWoWInstallFolder/Interface/AddOns/" Directory.
Now it looks like this: "yourWoWInstallFolder/Interface/AddOns/PowerAuras".
Type "/powa" ingame to open the options window.

1 = Turn the AddOn on or off.

2 = Auras only for this char.

3 = Auras for all your chars.

4 = Auras on this page.

5 = Create a new aura.

6 = Move an aura to other page

7 = Copy an aura to a page of your choice. Can be the same page.

8 = Deletes the marked aura. This only works when you press the CTRL-Key and click on the button.

9 = Opens the Effect Editor for the marked aura.
Ok, let's create a new aura. Click on the "New" Button.
The Effect Editor opens.
1 = Display of the texture you choosed.

2 = Color options

3 = The texture selector. "Glow" dark sections of the texture will become more transparent.
When you want an aura with dark color deactivate "Glow".
With "WoW Textures" u can choose from a selection of game graphics.
With "Custom Auras" u can enter a filename for your own textures.
These must be in "/AddOns/PowerAuras/Custom/" BEFORE u start the game. Example: "Flamme.tga"

4 = The display options for the selected texture.

5 = Position for the texture.
You can enter exact positions in the textfield right.

6 = The selection of the ingame event, that u want to trigger your aura.

7 = The icon of the buff/debuff/action. This is saved when the aura has worked once.

8 = The name of the buff/debuff/action. You can enter only a single word of the name or enter more names. See picture.

9 = The "object", which u want to check for the event. When u check nothing, the player is the "object".

10 = Displayoptions for the aura activation.
Ok, you created a new aura. Have choosed a texture, color, display und position.
Now u can selected an event that u want to check everytime.
For Example we take the HoT(HealthoverTime) "Renew" from a priest that heals you.
A HoT is a buff. We select "Buff" from the DropDown menu.
Now type in the textfield below "Renew".
Ok below that you have to check the "objects" that u want to ckeck for the event.
We check nothing, because we have the buff not anybody else.
And you can check that the aura is displayed then the event is on or off.
Or only when you are infight or outfight.

Now click on the "Animation" Tab. You see this:

1 = Beginning animation

2 = End animation

3 = Main animation

4 = Second animation. This stack with the main animation.

5 = Animation speed. You want a disco? Or some psychadelic lights?^^

6 = Animation duration. Is it "None", the aura is shown all the time that the event is on.
So enough of these. Go, get some auras on! :).
